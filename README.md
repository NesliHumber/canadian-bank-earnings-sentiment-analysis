# Canadian Bank Earnings Sentiment Analysis

## Objective

Comparative Sentiment and Topic Analysis of Canadian Bank Earnings Call Transcripts (2019-2026)

Banks:
- RBC
- TD
- Scotiabank

## Research Questions

1. How has management sentiment changed over time?
2. How did sentiment change during COVID-19?
3. What topics dominate earnings calls?
4. How do communication patterns differ among banks?

## Methods

- Text Cleaning
- VADER Sentiment Analysis
- TextBlob
- Topic Modeling (LDA)
- N-Grams
- Word Clouds

## Tools

Python, Pandas, NLTK, spaCy, Scikit-Learn, Gensim
